# Alias.Print-without-Running
Useful when you need to modify a Hard to Remeber command!  https://www.google.com/search?q=bash+alias+print+without+executing https://www.google.com/search?q=bash+alias+print+without+running https://www.google.com/search?q=bash+alias+write+prompt+without+executing https://askubuntu.com/questions/658087/create-an-alias-to-print-a-command-instead-of-executing-it

# Example:
## Git Tagging
```
alias gt="echo 'git tag -a v0.1 -m \"new version\"'"
alias gtp="git push origin --tags"
```
